# Ashish Maurya – Portfolio

Personal portfolio website of **Ashishkumar Maurya**, B.E. graduate in Artificial Intelligence & Data Science from VESIT Mumbai.

🌐 **Live:** [ashishmaurya.vercel.app](https://ashishmaurya.vercel.app)

## About

AI & ML Engineer with hands-on experience in computer vision, NLP, LLMs, and full-stack development. Built with Next.js, TypeScript, Tailwind CSS, and Framer Motion.

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Language:** TypeScript

## Getting Started
